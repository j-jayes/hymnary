## Methodology

### Input Data

The organ hymn list comes from an **Allen organ console controller display**, which shows 182 built-in hymns by abbreviated name. Each abbreviation was mapped to its full hymn title.

Some hymns have **A/B variants** (e.g., "Away in a Manger - A" and "Away in a Manger - B"), indicating that the organ has two different tune settings for the same text. In these cases, we search once and present all matching tunes, sorted by prevalence.

### Scraping Process

For each of the 182 organ hymns:

1. **Search** ‚Äî We queried `hymnary.org/search?qu={title}+in%3Atunes` to find tune matches.
2. **Tune detail** ‚Äî For each tune found in search results, we fetched the full detail page at `hymnary.org/tune/{slug}`.
3. **Parse** ‚Äî We extracted structured metadata using BeautifulSoup (Python) from the HTML:
   - Tune title, composer, place of origin
   - Meter pattern, melodic incipit, key signature
   - Copyright status, number of hymnal appearances
   - Associated texts and alternative tune settings
   - Historical notes where available
